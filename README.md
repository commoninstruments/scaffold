# Scaffold

Opinionated documentation scaffold for starting a new Howells project.

This repo is intentionally docs-first. It captures the shared baseline that is already emerging across recent active projects such as `patternmode`, `materia`, `sorrel`, `scenes`, `colorscope`, `curioda`, `populararchive`, plus the shared config repos `arc`, `lint`, `typescript-config`, and `turbo-config`.

Start with [docs/README.md](docs/README.md).
