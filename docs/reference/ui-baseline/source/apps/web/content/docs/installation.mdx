## Install packages

```bash
pnpm add @patternmode/ui @patternmode/motion @patternmode/tailwind-config
```

## Tailwind configuration

Import the shared styles in your main CSS file. This registers the full theme (colors, radii, shadows, typography) and base layer resets.

```css title="app/globals.css"
@import "@patternmode/tailwind-config";
```

If you need to source additional paths (e.g. components outside the default globs), add `@source` directives after the import.

## Font setup

PatternMode expects three CSS custom properties for font families. Set them on `<html>` or `<body>` via Next.js `next/font`:

```tsx title="app/layout.tsx"
import { Geist, Geist_Mono } from "next/font/google";
import localFont from "next/font/local";

const sans = Geist({ subsets: ["latin"], variable: "--font-sans" });
const mono = Geist_Mono({ subsets: ["latin"], variable: "--font-geist-mono" });
const serif = localFont({
  src: "./fonts/your-serif.woff2",
  variable: "--font-serif",
});

export default function RootLayout({ children }) {
  return (
    <html className={`${sans.variable} ${mono.variable} ${serif.variable}`}>
      <body>{children}</body>
    </html>
  );
}
```

The theme references these via `--font-family-sans`, `--font-family-serif`, and `--font-family-mono`.

## TypeScript paths (monorepo)

If you consume the packages as workspace dependencies, add path aliases in your `tsconfig.json`:

```json title="tsconfig.json"
{
  "compilerOptions": {
    "paths": {
      "@patternmode/ui/*": ["../../packages/ui/src/*"],
      "@patternmode/motion/*": ["../../packages/motion/src/*"],
      "@patternmode/tailwind-config": ["../../packages/tailwind-config/shared-styles.css"]
    }
  }
}
```
