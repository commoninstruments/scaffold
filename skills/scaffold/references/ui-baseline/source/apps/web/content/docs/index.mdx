PatternMode is a three-layer design system for building refined interfaces.

## Packages

**`@patternmode/ui`** -- Accessible, composable React components built on Radix primitives and styled with Tailwind CSS. Ships buttons, inputs, dialogs, data display, layout primitives, and more.

**`@patternmode/motion`** -- Physics-based spring configs, duration tokens, easing curves, and semantic presets for motion-react and Framer Motion.

**`@patternmode/transition`** -- Higher-order interaction patterns: shared element transitions, dismissible overlays, and coordinated sequences.

**`@patternmode/tailwind-config`** -- Shared Tailwind v4 theme with oklch color tokens, typography scale, radius, shadow elevation, and base styles.
