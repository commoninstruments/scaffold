title: Motion
description: Springs, durations, easings, and presets from @patternmode/motion.
---

`@patternmode/motion` provides tokens for physics-based and timed animations. All values are compatible with motion-react / Framer Motion.

```tsx
import { springs, durations, easings, presets } from "@patternmode/motion";
```

## Springs

Physics-based spring configs with `stiffness`, `damping`, and `mass`.

| Token | Stiffness | Damping | Mass | Use case |
|-------|----------:|--------:|-----:|----------|
| `soft` | 120 | 18 | 1 | Loaders, radial pickers |
| `subtle` | 300 | 30 | 1 | Professional, barely noticeable |
| `natural` | 200 | 20 | 1 | Default choice |
| `playful` | 170 | 15 | 1 | Character with bounce |
| `bouncy` | 260 | 12 | 1 | Popovers, hints |
| `snappy` | 400 | 28 | 0.8 | Quick interactions |
| `stiff` | 500 | 30 | 1 | Panels, drawers |
| `swift` | 500 | 30 | 0.5 | Dropdowns, zero overshoot |

```tsx
<motion.div transition={springs.natural} />
```

## Durations

Timed transitions in seconds (use `durationMs` for millisecond values).

| Token | Seconds | Ms | Use case |
|-------|--------:|---:|----------|
| `instant` | 0 | 0 | No animation |
| `quick` | 0.12 | 120 | Hovers, exits |
| `normal` | 0.25 | 250 | Dialogs, reveals |
| `moderate` | 0.4 | 400 | Page transitions |
| `slow` | 1.0 | 1000 | Shimmer, loaders |

## Easings

Cubic bezier curves as `[x1, y1, x2, y2]` tuples. CSS string variants available via `easingsCSS`.

| Token | Curve | Use case |
|-------|-------|----------|
| `smooth` | `(0.4, 0, 0.2, 1)` | General purpose |
| `customIn` | `(0.55, 0.085, 0.68, 0.53)` | Entering elements |
| `customOut` | `(0.165, 0.84, 0.44, 1)` | Exiting elements |
| `customExpand` | `(0.25, 0.46, 0.45, 0.94)` | Expansion |
| `customGentle` | `(0.25, 0.1, 0.12, 1)` | Gentle ease out |

## Scales

Transform scale multipliers for interactive feedback.

| Token | Value | Use case |
|-------|------:|----------|
| `press` | 0.97 | Subtle click shrink |
| `pressDeep` | 0.95 | Pronounced press |
| `hover` | 1.02 | Slight hover lift |
| `hoverLift` | 1.05 | Pronounced hover lift |

## Presets

Semantic combos that pair a duration or spring with an easing. Spread directly into the `transition` prop.

```tsx
import { presets } from "@patternmode/motion";

<motion.div
  initial={{ opacity: 0 }}
  animate={{ opacity: 1 }}
  transition={presets.fadeIn}
/>
```

| Preset | Composition |
|--------|-------------|
| `dialogOpen` | `normal` + `customOut` |
| `dialogClose` | `quick` + `customIn` |
| `slideIn` | `normal` + `customOut` |
| `slideOut` | `quick` + `customIn` |
| `fadeIn` | `quick` + `customOut` |
| `fadeOut` | `quick` + `customIn` |
| `hoverLift` | `snappy` spring |
| `hoverSettle` | `natural` spring |
| `shake` | `normal` + `customOut` |
