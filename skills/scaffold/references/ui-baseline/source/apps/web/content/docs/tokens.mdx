title: Design Tokens
description: oklch colors, radius, shadows, and typography scale.
---

All tokens are CSS custom properties defined in `@patternmode/tailwind-config`. They are available as Tailwind utilities automatically.

## Colors

The palette uses **oklch** for perceptual uniformity. Neutrals share hue 264 (blue-gray); the accent sits at hue 241 (saturated blue).

```css
/* Surfaces */
--color-background:       oklch(0.985 0.003 264)
--color-card:             oklch(1 0 0)
--color-surface-subtle:   oklch(0.975 0.003 264)
--color-surface-hover:    oklch(0.955 0.005 264)

/* Foreground */
--color-foreground:       oklch(0.141 0.02 264)
--color-foreground-subtle: oklch(0.446 0.016 264)
--color-foreground-faint: oklch(0.646 0.014 264)
--color-muted-foreground: oklch(0.556 0.016 264)

/* Accent */
--color-accent:           oklch(0.588 0.158 241)
--color-accent-subtle:    oklch(0.96 0.03 241)

/* Semantic */
--color-destructive:      oklch(0.577 0.245 27)
--color-success:          oklch(0.62 0.17 155)
--color-warning:          oklch(0.795 0.155 80)

/* Borders */
--color-border:           oklch(0.915 0.006 264)
--color-border-subtle:    oklch(0.94 0.004 264)
--color-border-strong:    oklch(0.85 0.012 264)
```

Dark mode overrides every token automatically via the `.dark` class.

## Radius

```css
--radius-sm: 0.375rem   /* 6px  — chips, tags */
--radius-md: 0.5rem     /* 8px  — inputs, buttons */
--radius-lg: 0.75rem    /* 12px — cards */
--radius-xl: 1rem       /* 16px — modals, sheets */
```

## Shadows

### Base scale

```css
--shadow-2xs  /* hairline — subtle borders */
--shadow-xs   /* resting cards */
--shadow-sm   /* hover lift */
--shadow-md   /* dropdowns */
--shadow-lg   /* modals */
```

### Elevation (semantic, multi-layer)

```css
--shadow-elevation-card-rest
--shadow-elevation-card-hover
--shadow-elevation-flyout
--shadow-elevation-modal
--shadow-elevation-tooltip
--shadow-elevation-commandbar
```

### Button-specific

```css
--shadow-buttons-neutral
--shadow-buttons-inverted
--shadow-buttons-danger
--shadow-outline-base
```

## Typography scale

```css
/* Label — caps, small tags */
--text-label:      0.75rem / 1rem      (letter-spacing: 0.06em)

/* Body */
--text-body:       0.9375rem / 1.5rem  (letter-spacing: -0.01em)
--text-body-lg:    1.0625rem / 1.625rem (letter-spacing: -0.012em)

/* Titles */
--text-title-sm:   1.375rem / 1.8rem   (letter-spacing: -0.025em)
--text-title:      2.25rem / 2.6rem    (letter-spacing: -0.035em)
--text-title-lg:   3rem / 3.3rem       (letter-spacing: -0.04em)
```

### Letter spacing utilities

```css
--tracking-tight:   -0.025em
--tracking-display: -0.035em
--tracking-body:    -0.012em
```
